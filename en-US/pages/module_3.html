Course III

Flask

Creating Web Applications


Flask Framework provides a collection of tools that a developer uses when creating a web application. In this course, the students will synthesize the knowledge gained in the previous courses (Python, HTML, and CSS). Students will create basic Web Apps and a web-scraping bot. This course includes presentation of theoretical material, providing students with multiple opportunities to practice skills by solving real life tasks, and assessing students’ knowledge.


Concepts covered in this course:

  1. Flask Installation
  2. Command Line
  3. Flask templates
  4. Flask forms
  5. JSON file
  6. Libraries
  7. Web Scraping
