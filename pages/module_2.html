Модуль II

HTML and CSS

HTML (HyperText Markup Language) – язык, который позволяет структурировать веб-страницу. HTML дает браузеру понять, как отображать информацию на веб-странице. CSS (Cascading Style Sheets) - язык, который используется для стилизации HTML-документа. В модуль входит: презентация теоретического материала (слайды на каждом занятии), решение практических задач, которые подобраны с учетом интересов студентов, а также оценка знаний студентов (проектные работы).

CSS3

Модуль включает в себя следующие темы:

  1. HTML elements
  2. HTML table
  3. HTML forms
  4. CSS Selectors
  5. CSS Properties
  6. CSS Box
