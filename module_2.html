Course II

HTML and CSS


HTML5

CSS3

HTML (HyperText Markup Language) is the language that provides structure to the content appearing on a website, such as images, text, or videos. CSS (Cascading Style Sheets) is the language that is used to style HTML content on a web page. This course includes presentation of theoretical material, providing students with multiple opportunities to practice skills by solving real life tasks, and assessing students’ knowledge.


Concepts covered in this course:

  1. HTML elements
  2. HTML table
  3. HTML forms
  4. CSS Selectors
  5. CSS Properties
  6. CSS Box
