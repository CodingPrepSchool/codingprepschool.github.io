Модуль IV

Основы Git. Github.


Github

Git - бесплатное программное обеспечение, которое позволяет программистам отслеживать изменения в проекте и работать совместно над одним проектом. Github позволяет программистам сохранить окончательный вариант проекта и создать профессиональное портфолио.


Модуль включает в себя работу над созданием веб-сайта с использованием основных команд Git: clone, init, add, commit, push, log, diff, status, checkout, reset.
